# Changelog

모든 주요 변경을 이 파일에 기록합니다. 형식은 [Keep a Changelog](https://keepachangelog.com/ko/),
버전은 [유의적 버전(SemVer)](https://semver.org/lang/ko/)을 따릅니다.
이 파일은 앱의 "업데이트 내역"에도 그대로 표시됩니다.

## [0.6.0] - 2026-06-16

### 새 기능
- **노트 ↔ 노트 연결** — 본문에서 `[[노트 제목]]`으로 노트를 잇습니다. `[[` 입력 시 자동완성이 뜨고, 없는 노트는 그 자리에서 만들어 연결합니다. 링크를 누르면 **미리보기(peek)**가 떠 보던 노트를 잃지 않고, "열기"나 **⌘-클릭**으로 반대쪽 화면에 나란히 엽니다. 아직 없는 노트로의 링크는 흐리게 표시됩니다.
- **백링크(양방향)** — 노트 제목 옆 "정보"에서 *이 노트를 참조하는 노트들*을 봅니다. 한쪽만 링크해도 양쪽에서 오갈 수 있습니다.
- **노트 "정보" 패널** — 목차 · 연결된 노드 · 백링크를 한곳에 모았습니다(기본 닫힘, 있을 때만 표시).
- **노드 순서 드래그** — 형제 노드를 끌어 순서를 바꿉니다(위/아래 틈에 놓으면 형제로 삽입, 가운데에 놓으면 자식으로).

### 개선
- **탭 순서 드래그** — 탭을 끌어 순서를 바꾸고 좌우로 옮길 수 있습니다(드래그가 되지 않던 문제 해결).

### 버그 수정
- 노드에서 "새 노트 만들어 연결"이 조용히 실패하던 문제 — 이제 실패 시 알려주고, 한 단계가 실패해도 나머지는 진행됩니다.
- 노트 이름을 바꾸면 다른 노트의 `[[ ]]` 링크가 깨지던 문제 — 참조하는 모든 노트에서 자동으로 갱신됩니다.

## [0.5.3] - 2026-06-15

### 새 기능
- **노트 목차** — 헤딩이 2개 이상인 노트 맨 위에 "목차" 바가 생깁니다. 펼쳐서 항목을 누르면 그 헤딩으로 바로 이동합니다.

## [0.5.2] - 2026-06-15

### 새 기능
- **업데이트 내역** — 새 버전으로 업데이트하면 "새로운 점" 카드로 바뀐 점을 알려주고, 설정에서 전체 버전의 변경점을 언제든 볼 수 있습니다.
- **노트 표 행/열 편집** — 표 안에 커서를 두면 행·열을 추가·삭제할 수 있습니다. (열 너비 조정은 마크다운 표준에 없어 제외)

### 버그 수정
- 사이드바에서 **노트 제목을 바꾸면** 편집 패널과 파일명이 함께 바뀌고, 다시 옛 제목으로 되돌아가던 문제를 고쳤습니다.

## [0.5.1] - 2026-06-14

### 버그 수정
- **노드에서 만든 노트가 사이드바에 보입니다** — 숨김 폴더로 들어가던 문제 수정(이제 노드의 맵과 같은 폴더에 생성). 이전 버전에서 숨어 있던 노드 노트도 앱을 열 때 자동으로 복구됩니다.

### 개선
- 사이드바에서 숨기는 건 집중 세션(work-log) 노트뿐 — 사용자가 만든 "work-log" 폴더는 보입니다.

## [0.5.0] - 2026-06-12

### 새 기능
- **집중 세션** — 노드에서 집중 세션을 시작해 목표·과정·결과를 남깁니다. 타이머, 작업 로그 노트, 완료 카드.
- **"오늘" 뷰** — 여러 맵에 흩어진 일정을 한 화면에 모으고 클릭 한 번에 집중 시작.
- **돌아보기** — 집중 기록을 오늘 ↔ 이번 주로 회고.
- **노트 표(테이블)** — GFM 마크다운.

### 개선
- 접근성(WCAG AA), 디자인 일관성, 사이드바 내비 정리, 크래시 진단 로깅.

### 버그 수정
- 집중 목표 추출 오류, 완료 카드 회고 보존, 대시보드 인터랙션 결함.
