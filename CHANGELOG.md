# Changelog

모든 주요 변경을 이 파일에 기록합니다. 형식은 [Keep a Changelog](https://keepachangelog.com/ko/),
버전은 [유의적 버전(SemVer)](https://semver.org/lang/ko/)을 따릅니다.
이 파일은 앱의 "업데이트 내역"에도 그대로 표시됩니다.

## [0.5.2] - 2026-06-15

### 새 기능
- **업데이트 내역** — 새 버전으로 업데이트하면 "새로운 점" 카드로 바뀐 점을 알려주고, 설정에서 전체 버전의 변경점을 언제든 볼 수 있습니다.
- **노트 표 행/열 편집** — 표 안에 커서를 두면 행·열을 추가·삭제할 수 있습니다. (열 너비 조정은 마크다운 표준에 없어 제외)

### 버그 수정
- 사이드바에서 **노트 제목을 바꾸면** 편집 패널과 파일명이 함께 바뀌고, 다시 옛 제목으로 되돌아가던 문제를 고쳤습니다.

## [0.5.1] - 2026-06-14

### 버그 수정
- **노드에서 만든 노트가 사이드바에 보입니다** — 숨김 폴더로 들어가던 문제 수정(이제 노드의 맵과 같은 폴더에 생성). 이전 버전에서 숨어 있던 노드 노트도 앱을 열 때 자동으로 복구됩니다.

### 개선
- 사이드바에서 숨기는 건 집중 세션(work-log) 노트뿐 — 사용자가 만든 "work-log" 폴더는 보입니다.

## [0.5.0] - 2026-06-12

### 새 기능
- **집중 세션** — 노드에서 집중 세션을 시작해 목표·과정·결과를 남깁니다. 타이머, 작업 로그 노트, 완료 카드.
- **"오늘" 뷰** — 여러 맵에 흩어진 일정을 한 화면에 모으고 클릭 한 번에 집중 시작.
- **돌아보기** — 집중 기록을 오늘 ↔ 이번 주로 회고.
- **노트 표(테이블)** — GFM 마크다운.

### 개선
- 접근성(WCAG AA), 디자인 일관성, 사이드바 내비 정리, 크래시 진단 로깅.

### 버그 수정
- 집중 목표 추출 오류, 완료 카드 회고 보존, 대시보드 인터랙션 결함.

## [0.4.1] - 2026-06-12

### 버그 수정
- 미리알림 동기화를 항목별·필드별로 맞춰 양쪽 변경을 모두 보존.
- 노트 슬래시(`/`) 메뉴가 스크롤 시 허공에 남던 문제.

## [0.4.0] - 2026-06-12

### 새 기능
- **노트 이미지** — 붙여넣기·드래그·툴바로 삽입, 자동 축소 후 노트에 내장.

### 버그 수정
- 노트 제목을 바꿔도 사이드바·탭이 그대로이던 문제.

## [0.3.0] - 2026-06-11

### 새 기능
- **자동 업데이트** — 새 버전을 백그라운드에서 받고 "지금 재시동 / 나중에" 안내.

## [0.2.1] - 2026-06-11

### 버그 수정
- 동명 폴더로 이동/승격 시 노트가 사라져 보이던 문제 외 코드리뷰 7건. 이 버전부터 Apple 공증 서명 빌드.

## [0.2.0] - 2026-06-11

### 새 기능
- **노트 시스템**(독립 마크다운·노드 양방향 연결·URL 추출), 제목 자동 이름, 시맨틱 색상, 화면 분할, 앱 아이콘.

### 개선
- 미리알림 인바운드 동기화 복구(23초 → 0.85초), 우클릭 메뉴 5그룹.

### 버그 수정
- 노트 픽커 Esc 미동작, 우클릭 메뉴 가림, 공백 노드 잔존.
