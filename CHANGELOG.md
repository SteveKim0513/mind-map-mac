## [0.8.3] - 2026-07-15

### 버그 수정
- **노트 이미지 첨부 폴더가 Finder 등 앱 밖에서는 숨겨지지 않던 문제 수정** — `notename.assets/` 폴더가 앱 사이드바에서만 걸러지고 실제로는 점(`.`)으로 시작하지 않아 외부에서 그대로 노출됐다. `.notename.assets/`로 이름을 바꿔 OS 표준 숨김 규칙을 타도록 했다. 기존 폴더도 계속 인식되며, 이름 변경·이동·휴지통 이동·복원 시점에 자동으로 숨김 이름으로 승격된다.

## [0.8.2] - 2026-07-14

### 버그 수정
- **자연어 일정 인식에 "@" 마커를 필수화** — "오늘"/"내일" 등이 문장 어디서든(라벨 포함) 매칭되던 방식이 "오늘 : 회의 메모"처럼 날짜 단어를 라벨로 쓰는 습관과 자주 충돌해 의도치 않은 일정이 계속 생겼다. `#색상`과 대칭되는 "@내일 3시" 형태로 마커를 필수화.
- **리스트 내어쓰기(Shift+Tab) 시 뒤따르는 형제 항목이 함께 딸려가던 문제 수정** — 내어쓰는 항목 뒤에 같은 깊이의 형제가 더 있으면 그 형제가 방금 내어쓴 항목의 자식으로 재배치되던 ProseMirror 기본 동작을 수정. 이제 뒤 형제는 원래 부모 밑에 그대로 남고, 내어쓰는 항목만 옮겨진다.

## [0.8.1] - 2026-07-13

### 버그 수정
- **전역 캡처(⌥Space) 대상 맵이 탭으로 열려 있으면 캡처가 사라지던 문제 수정** — 열린 탭의 인메모리 상태가 stale한 채로 다음 자동저장 때 디스크의 캡처를 덮어썼다(맵을 보기만 해도 발생 가능). 캡처 저장 후 열려 있는 탭을 디스크에서 다시 읽어오도록 수정.
- **노트 리네임 후 임베드 이미지가 깨지던 문제 수정** — 제목에 붙은 보이지 않는 제어문자(paste로 유입) 때문에 파일명이 오염되고, 리네임 시 이미지 폴더는 새 이름으로 바뀌지만 본문의 이미지 링크는 예전 이름을 그대로 참조해 깨졌다. 제목에서 제어문자를 제거하고, 리네임 시 본문의 이미지 링크도 함께 재작성하도록 수정.
- **탭을 전환했다 돌아오면 노트 스크롤 위치가 맨 위로 초기화되던 문제 수정.**
- **일정 팝오버의 "스케줄 지우기"가 실제로는 일정을 지우지 않던 문제 수정** — 날짜만 비우고 "일정 등록됨" 상태는 그대로 남아 칩이 계속 표시됐다. 이제 완전히 지워지며, 연결된 리마인더도 함께 정리된다.

## [0.8.0] - 2026-07-13

### 신규
- **전역 즉시 캡처 (⌥Space)** — 앱이 백그라운드에 있거나 메인 창이 닫혀 있어도 전역 단축키로 작은 캡처 창이 뜨고, Enter로 워크스페이스의 "오늘의 생각.mind"에 새 루트 노드로 즉시 쌓입니다. ⌘K "빠른 캡처 열기" 명령으로도 접근 가능. 단축키가 다른 앱과 충돌하면 조용히 비활성화되고 설정 → 고급 설정에 상태가 표시됩니다.
- **자연어 일정·태그 인식** — 노드에 "내일 3시", "다음 주 월요일" 같은 문구를 입력하면 자동으로 일정이 잡히고, `#태그명`(영문 키 또는 한글 별칭)을 입력하면 자동으로 색이 지정됩니다. 날짜·요일 표현이 없으면 절대 매칭하지 않아 일반 문장과 헷갈리지 않습니다. 전역 캡처에도 동일하게 적용됩니다.
- **Smart Views: 최근 수정 · 즐겨찾기** — 사이드바 상단에 "최근 수정"(파일 수정 시각순)과 "즐겨찾기"(파일에 마우스를 올려 별표로 추가/해제) 두 뷰가 추가됐습니다.
- **⌘K 자동 학습 단축키** — 명령 팔레트에서 같은 명령을 2회 이상 실행하면 목록 맨 위로 올라오고 `⌥1`~`⌥9` 단축키가 자동으로 배정됩니다.

### 개선
- 설정 화면이 테마·워크스페이스·노트 템플릿 3개만 기본으로 보이고, 나머지(AI·정보 양식·사용 안내·업데이트)는 "고급 설정 보기"를 눌러야 나타납니다.
- 노트 편집기의 "메타+" 버튼과 설정의 "메타 템플릿"이 각각 "양식+"/"정보 양식"으로 이름이 바뀌었습니다.
- 사이드바에 독킹된 집중 세션 위젯이 기본적으로 작은 타이머만 보이는 형태로 접히고, 클릭하면 펼쳐집니다.
- 휴지통·Note Template 아이콘의 숫자 배지가 "몇 개 있는지"보다 "뭔가 있는지"만 알려주는 조용한 점 표시로 바뀌었습니다.
- 우클릭 메뉴·메타/템플릿 드롭다운 등 플로팅 UI의 그림자·등장 애니메이션을 공유 토큰으로 정리했습니다.

## [0.7.33] - 2026-07-10

### 신규
- **노트 템플릿** — 회의록·회고 등 자주 쓰는 노트 구조를 템플릿으로 저장해두고 편집기 툴바 "템플릿+"(검색 가능)로 커서 위치에 바로 삽입. 템플릿은 워크스페이스의 숨김 시스템 폴더 `.templates/`에 저장되며 삭제해도 자동 재생성됨. 사이드바 하단 "Note Template" 버튼으로 목록·생성·삭제 관리. 설정에서 기능 자체를 켜고 끌 수 있음. 기존 "메타+" 드롭다운에도 검색창 추가.

### 보안
- **워크스페이스 경계 미검증 IPC 핸들러 하드닝** — `fs:createFile`/`fs:createFolder`/`fs:rename`/`fs:delete`/`fs:move`/`images:write`/`images:read`가 입력 경로를 검증하지 않아, 노트·맵 제목에 `../`를 넣으면 워크스페이스 밖에 파일을 쓸 수 있었던 문제를 수정. 공용 `assertInsideWorkspace` 검증을 전 핸들러에 적용.

### 내부
- E2E 플레이키 테스트 수정 — 외부 파일 추가 후 포커스 복귀 시 사이드바 갱신 테스트가 고정 sleep 대신 실제 상태를 폴링하도록 변경.

- **백링크(양방향)** — 노트 제목 옆 "정보"에서 *이 노트를 참조하는 노트들*을 봅니다. 한쪽만 링크해도 양쪽에서 오갈 수 있습니다.
- **노트 "정보" 패널** — 목차 · 연결된 노드 · 백링크를 한곳에 모았습니다(기본 닫힘, 있을 때만 표시).
- **노드 순서 드래그** — 형제 노드를 끌어 순서를 바꿉니다(위/아래 틈에 놓으면 형제로 삽입, 가운데에 놓으면 자식으로).

### 개선
- **탭 순서 드래그** — 탭을 끌어 순서를 바꾸고 좌우로 옮길 수 있습니다(드래그가 되지 않던 문제 해결).

### 버그 수정
- 노드에서 "새 노트 만들어 연결"이 조용히 실패하던 문제 — 이제 실패 시 알려주고, 한 단계가 실패해도 나머지는 진행됩니다.
- 노트 이름을 바꾸면 다른 노트의 `[[ ]]` 링크가 깨지던 문제 — 참조하는 모든 노트에서 자동으로 갱신됩니다.
