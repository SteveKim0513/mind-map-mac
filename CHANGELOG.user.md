# 새로운 소식

> 앱의 "업데이트 내역"에 보이는, 고객을 위한 변경 요약입니다.
> (개발용 상세 변경은 `CHANGELOG.md`를 따로 둡니다.)

## [0.7.35] - 2026-07-10

- **⌘K로 노드까지 찾을 수 있습니다.** 명령 팔레트(⌘K)에 검색어를 입력하면 열려 있는 맵의 노드도 함께 나옵니다. 더 찾고 싶으면 하단의 "전체에서 찾기"를 눌러 노트까지 포함한 전체 검색으로 바로 이어갈 수 있어요.
- **필요할 때 살짝 알려드립니다.** 탭을 많이 열었거나, 비슷한 노트를 여러 번 쓰거나, 한 맵에 일정이 늘어나면 화면 분할·템플릿·리마인더 동기화를 한 번씩 제안해 드려요. 원치 않으면 그냥 닫으면 다시 뜨지 않습니다.
- 사이드바와 노트 편집기 툴바를 조금 더 정리했습니다 (아이콘 구분, 그룹핑).

## [0.7.34] - 2026-07-10

- **노트 템플릿 탭을 일반 노트와 헷갈리지 않게 됩니다.** 템플릿 탭은 이제 전용 아이콘으로 표시되고, 편집 화면 제목 옆에도 "템플릿" 표시가 항상 붙어 있어 여러 탭을 열어둬도 바로 구분됩니다.
- **템플릿 만들기 버튼이 어디서 눌러도 똑같이 동작합니다.** 툴바에서 "새 템플릿 만들기"를 눌러도 사이드바에서 만든 것처럼 바로 편집 화면이 열립니다.

## [0.7.33] - 2026-07-10

- **노트 템플릿 기능이 추가되었습니다.** 회의록·회고처럼 자주 쓰는 노트 구조를 템플릿으로 저장해두면, 다른 노트를 쓸 때 편집기 툴바의 "템플릿+" 버튼(검색 가능)으로 커서 위치에 바로 불러올 수 있습니다. 사이드바 하단의 "Note Template" 버튼에서 템플릿을 만들고 관리하며, 설정에서 이 기능을 완전히 끌 수도 있습니다.
- 노트 편집기의 "메타+" 버튼에도 검색창이 추가되어, 항목이 많아져도 빠르게 찾을 수 있습니다.
- 파일 저장 경로 관련 보안을 강화했습니다.

## [0.7.32] - 2026-07-09

- **창 폭이 좁아질 때 텍스트가 세로로 깨지던 문제를 고쳤습니다.** 노트 툴바의 "메타+" 버튼, 빠른 열기·전체 검색·명령 팔레트 목록, 메타 블록 제목 등이 창을 좁히거나 화면을 분할했을 때 글자가 한 줄에 하나씩 세로로 늘어서던 현상이 사라지고, 이제 자연스럽게 줄여 표시되거나 스크롤됩니다.

## [0.7.31] - 2026-07-09

- **내부 안정성 개선.** 배포 전 전체 E2E 테스트 통과가 의무화되어 회귀 버그가 더 빠르게 잡힙니다.

## [0.7.30] - 2026-07-09

- **메타 템플릿을 삭제하면 노트에서 자동으로 사라집니다.** 설정에서 템플릿을 삭제하면 해당 템플릿을 사용 중인 모든 노트의 메타 블록이 즉시 제거됩니다. "삭제된 템플릿" 경고가 남지 않습니다.

## [0.7.29] - 2026-07-09

- **메타 값이 검색에서 찾아집니다.** 노트에 입력한 메타 블록 값을 전역 검색(⌘K)에서 바로 검색할 수 있습니다. 메타로 매칭된 결과는 "메타: [값]"으로 표시됩니다.
- **메타+ 드롭다운이 잘리던 오류를 수정했습니다.** 툴바가 좁을 때 메타+ 버튼의 드롭다운이 잘려 보이거나 스크롤이 생기던 문제를 수정했습니다.

## [0.7.28] - 2026-07-09

- **설정 화면이 더 깔끔해졌습니다.** AI 기능, 메타 템플릿, 업데이트 내역이 각각 독립된 서브 화면으로 분리되었습니다. 설정 메인에서 항목을 탭하면 전용 화면으로 이동하고, 뒤로 버튼 또는 Esc로 돌아옵니다.
- **업데이트 내역이 설정 뒤에 가려지던 오류를 수정했습니다.**

## [0.7.27] - 2026-07-09

- **메타 템플릿 편집기의 여러 버그를 수정했습니다.** 선택(select) 필드의 옵션 입력 영역이 옆에 붙어 나오던 레이아웃 문제, 옵션 입력 시 쉼표나 띄어쓰기가 입력되지 않던 문제를 수정했습니다.
- **메타 블록 안정성이 개선되었습니다.** 파일이 손상된 경우에도 크래시 없이 안전하게 열리고, 이름 없이 템플릿을 저장하는 실수를 방지합니다.

## [0.7.26] - 2026-07-09

- **노트에 구조화된 메타 정보를 추가할 수 있습니다.** 설정(⌘,) → 메타 템플릿에서 text·date·select·url·number 타입의 필드를 자유롭게 정의하고, 노트 툴바 "메타+" 버튼으로 원하는 템플릿을 선택해 노트 상단에 삽입합니다.
- **select 필드는 드롭다운, date 필드는 날짜 피커로 입력합니다.** 템플릿을 수정하면 해당 템플릿을 사용 중인 노트에 자동으로 반영됩니다. 삭제된 필드는 값을 보존한 채 경고만 표시합니다.

## [0.7.25] - 2026-07-08

- **사이드바가 더 직관적으로 바뀌었습니다.** 오늘·돌아보기가 파일 목록 위에 탐색 항목으로 올라오고, 마인드맵·노트·URL 가져오기·새 폴더 버튼이 라이브러리 헤더에 한 줄로 모였습니다.
- **화면 곳곳의 표현이 더 자연스러워졌습니다.** "URL에서 가져오기", "연결 안 됨", "연결 끊기", "영구 삭제" 등 딱딱하거나 어색했던 용어를 다듬었습니다.
- **API 키 입력 중 취소할 수 있습니다.** 설정에서 API 키를 추가하다가 취소해도 이전 상태로 돌아옵니다.

## [0.7.24] - 2026-07-06

- **Claude와 GPT API 키를 모두 설정할 수 있습니다.** 설정(⌘,) → AI 기능에서 Claude(Anthropic)와 GPT(OpenAI) 키를 각각 등록할 수 있습니다. 키를 발급받는 링크도 설정 화면에서 바로 열립니다.
- **두 키가 모두 등록되면 하나를 선택해 활성화할 수 있습니다.** 라디오 버튼으로 어떤 모델을 사용할지 전환합니다. 하나만 등록하면 자동으로 활성화됩니다.

## [0.7.23] - 2026-07-06

- **설정에서 Claude API 키를 입력하면 AI 기능이 활성화됩니다.** ⌘, → AI 기능 섹션에서 Anthropic API 키를 등록하면 앞으로 추가될 AI 기능(태그 추천 등)을 사용할 수 있습니다. 키는 macOS 키체인에 암호화 저장됩니다.

## [0.7.22] - 2026-07-02

- **목록에서 Tab 들여쓰기가 안정적으로 동작합니다.** 첫 번째 항목에서 탭 문자가 삽입되거나 툴바로 포커스가 이동하던 문제를 수정했습니다.

## [0.7.21] - 2026-07-02

- **노트에서 블록을 드래그해 순서를 바꿀 수 있습니다.** 단락·헤더·목록 등 왼쪽에 ⠿ 핸들이 나타나고, 드래그해서 위아래로 이동합니다. ⌘Z로 되돌릴 수 있습니다.

## [0.7.20] - 2026-07-02

- **불릿/체크리스트에서 Tab 들여쓰기가 항상 동작합니다.** 기존 내용이 있는 항목에서 탭 문자가 삽입되던 문제를 고쳤습니다.
- **파일 여러 개를 선택해 폴더로 한 번에 드래그할 수 있습니다.** 사이드바에서 복수 선택 후 폴더 위로 드래그하면 모두 이동됩니다.
- **툴바로 블록 타입을 바꾸면 항상 올바르게 전환됩니다.** 불릿→인용처럼 다른 타입으로 전환이 안 되던 문제를 고쳤습니다.

## [0.7.19] - 2026-06-26

- **⌘W로 탭을 닫을 수 있습니다.**
- **링크로 노트 가져올 때 같은 이름 노트가 이미 있으면 기존 노트로 이동합니다.** 중복 파일이 생기지 않습니다.
- **파일 이름을 바꿀 때 같은 이름이 있으면 알려줍니다.** 기존에는 조용히 "이름 2"로 만들었는데, 이제 "이름의 파일이 이미 있습니다" 메시지가 표시됩니다.

## [0.7.18] - 2026-06-26

- **링크로 노트 가져올 때 같은 노트가 2개 생기던 문제를 고쳤습니다.** 링크에서 가져온 제목에 공백이 2개 이상 연속으로 있으면 파일명이 공백 1개인 기존 노트와 달라, 자동 이름변경이 충돌해 중복 파일이 만들어지던 현상입니다.

## [0.7.17] - 2026-06-24

- **긴 제목의 노트가 갑자기 2개로 복제되던 문제를 고쳤습니다.** 파일명 한도(60자)보다 긴 이름의 노트를 열면 자동 이름변경이 실행되어 기존 파일과 충돌, 원본과 동일한 내용의 중복 파일이 생기는 현상입니다.

## [0.7.16] - 2026-06-24

- **"정보" 패널이 "개요"로 바뀌었습니다.** 제목·연결·백링크를 한눈에 확인하는 패널의 이름이 더 직관적으로 변경됩니다.
- **목차가 긴 노트에서 에디터 영역이 눌리던 문제를 고쳤습니다.** 개요 패널이 화면 절반을 초과하면 자체적으로 스크롤되어 노트 편집 영역이 항상 확보됩니다.
- **섹션 레이블에 항목 수가 표시됩니다.** 목차 몇 개, 백링크 몇 개인지 한눈에 알 수 있습니다.

## [0.7.15] - 2026-06-24

- **이미지 관련 여러 문제를 고쳤습니다.**
  - 툴바 버튼으로 삽입한 이미지가 외부 파일로 저장되지 않던 문제를 고쳤습니다.
  - 노트 미리보기 팝업에서 이미지가 보이지 않던 문제를 고쳤습니다.
  - 노트 이름을 바꾸거나, 폴더로 이동하거나, 휴지통에 넣었다 복원하면 이미지가 사라지던 문제를 고쳤습니다.
  - 같은 초에 이미지를 여러 장 붙여넣으면 파일이 겹치던 문제를 고쳤습니다.
  - 이미지 보관 폴더(`.assets`)가 사이드바에 노출되던 문제를 고쳤습니다.

## [0.7.14] - 2026-06-24

- **이미지를 클릭하면 크게 볼 수 있습니다.** 노트 안의 이미지를 클릭하면 전체 화면으로 확대됩니다. ESC 또는 바깥 클릭으로 닫습니다.
- **이미지 화질이 좋아졌습니다.** 최대 해상도가 올라가고 JPEG 압축이 덜해져 선명하게 표시됩니다.
- **이미지가 별도 파일로 저장됩니다.** `노트이름.assets/` 폴더에 저장되어 마크다운 파일 크기가 작아지고, Obsidian 등 다른 에디터에서도 이미지가 보입니다.

## [0.7.13] - 2026-06-23

- **코드블록 언어 선택이 안 되던 문제를 고쳤습니다.** 언어 드롭다운을 누르면 메뉴가 바로 닫히던 현상입니다.

## [0.7.12] - 2026-06-23

- **노트에 코드블록이 생겼습니다.** `/` 메뉴나 툴바의 `{ }` 버튼으로 추가하고(또는 줄 맨 앞에 백틱 3개), 언어를 고르면 **문법 강조**가 됩니다. 흔한 언어(JS·TS·Python·HTML·CSS·JSON·SQL·Bash·Go·Rust·Java 등) 지원 + 자동 감지·코드 복사 버튼.

## [0.7.11] - 2026-06-23

- **휴지통이 생겼습니다.** 파일을 삭제하면 즉시 사라지지만 휴지통으로 들어가고, 사이드바 하단의 휴지통에서 **복원**하거나 **비우기**로 완전히 삭제할 수 있어요. (삭제 직후 "실행 취소"로도 바로 되돌릴 수 있습니다.)

## [0.7.10] - 2026-06-23

- **삭제한 파일이 사이드바에 그대로 남던 문제를 고쳤습니다.** 삭제 직후 자동 저장이 파일을 되살려, 탭은 닫혔는데 목록엔 남아 있던 현상입니다.

## [0.7.9] - 2026-06-23

- **자동 업데이트가 멈추던 문제를 고쳤습니다.** 일부 버전이 자신을 "개발 빌드"로 잘못 인식해 "업데이트 확인"이 꺼져 있던 문제입니다. 이제 새 버전을 정상적으로 받아옵니다. (이전 버전을 쓰던 분은 한 번만 새로 설치하면 이후 업데이트는 자동으로 됩니다.)

## [0.7.8] - 2026-06-22

- **삭제한 파일이 다시 나타나던** 문제를 고쳤습니다.
- **삭제가 실패하면** 이제 알림이 뜨고 파일·탭이 그대로 유지됩니다.
- 노트 미리보기(peek) 팝업이 열린 상태에서 해당 파일을 삭제하면 팝업도 함께 닫힙니다.

## [0.7.7] - 2026-06-22

- 파인더에서 파일을 추가·삭제·이동한 뒤 앱으로 돌아오면 **사이드바가 자동으로 업데이트**됩니다.

## [0.7.6] - 2026-06-22

- **이미 삭제된 노트**를 열려 할 때 탭이 그대로 남아있던 문제를 고쳤습니다.

## [0.7.5] - 2026-06-22

- **링크로 노트 만들기**를 할 때 동일한 노트가 **여러 개 생성되던** 문제를 고쳤습니다.

- **체크리스트**가 다른 마크다운 앱에서도 깔끔하게 보이도록 정리했습니다.

## [0.7.3] - 2026-06-17

- 노드를 **여러 개 선택하고 우클릭**하면, 여러 개에 한 번에 쓸 수 있는 기능만 깔끔하게 보여줍니다.
