# 새로운 소식

> 앱의 "업데이트 내역"에 보이는, 고객을 위한 변경 요약입니다.
> (개발용 상세 변경은 `CHANGELOG.md`를 따로 둡니다.)

## [0.7.15] - 2026-06-24

- **이미지 관련 여러 문제를 고쳤습니다.**
  - 툴바 버튼으로 삽입한 이미지가 외부 파일로 저장되지 않던 문제를 고쳤습니다.
  - 노트 미리보기 팝업에서 이미지가 보이지 않던 문제를 고쳤습니다.
  - 노트 이름을 바꾸거나, 폴더로 이동하거나, 휴지통에 넣었다 복원하면 이미지가 사라지던 문제를 고쳤습니다.
  - 같은 초에 이미지를 여러 장 붙여넣으면 파일이 겹치던 문제를 고쳤습니다.
  - 이미지 보관 폴더(`.assets`)가 사이드바에 노출되던 문제를 고쳤습니다.

## [0.7.14] - 2026-06-24

- **이미지를 클릭하면 크게 볼 수 있습니다.** 노트 안의 이미지를 클릭하면 전체 화면으로 확대됩니다. ESC 또는 바깥 클릭으로 닫습니다.
- **이미지 화질이 좋아졌습니다.** 최대 해상도가 올라가고 JPEG 압축이 덜해져 선명하게 표시됩니다.
- **이미지가 별도 파일로 저장됩니다.** `노트이름.assets/` 폴더에 저장되어 마크다운 파일 크기가 작아지고, Obsidian 등 다른 에디터에서도 이미지가 보입니다.

## [0.7.13] - 2026-06-23

- **코드블록 언어 선택이 안 되던 문제를 고쳤습니다.** 언어 드롭다운을 누르면 메뉴가 바로 닫히던 현상입니다.

## [0.7.12] - 2026-06-23

- **노트에 코드블록이 생겼습니다.** `/` 메뉴나 툴바의 `{ }` 버튼으로 추가하고(또는 줄 맨 앞에 백틱 3개), 언어를 고르면 **문법 강조**가 됩니다. 흔한 언어(JS·TS·Python·HTML·CSS·JSON·SQL·Bash·Go·Rust·Java 등) 지원 + 자동 감지·코드 복사 버튼.

## [0.7.11] - 2026-06-23

- **휴지통이 생겼습니다.** 파일을 삭제하면 즉시 사라지지만 휴지통으로 들어가고, 사이드바 하단의 휴지통에서 **복원**하거나 **비우기**로 완전히 삭제할 수 있어요. (삭제 직후 "실행 취소"로도 바로 되돌릴 수 있습니다.)

## [0.7.10] - 2026-06-23

- **삭제한 파일이 사이드바에 그대로 남던 문제를 고쳤습니다.** 삭제 직후 자동 저장이 파일을 되살려, 탭은 닫혔는데 목록엔 남아 있던 현상입니다.

## [0.7.9] - 2026-06-23

- **자동 업데이트가 멈추던 문제를 고쳤습니다.** 일부 버전이 자신을 "개발 빌드"로 잘못 인식해 "업데이트 확인"이 꺼져 있던 문제입니다. 이제 새 버전을 정상적으로 받아옵니다. (이전 버전을 쓰던 분은 한 번만 새로 설치하면 이후 업데이트는 자동으로 됩니다.)

## [0.7.8] - 2026-06-22

- **삭제한 파일이 다시 나타나던** 문제를 고쳤습니다.
- **삭제가 실패하면** 이제 알림이 뜨고 파일·탭이 그대로 유지됩니다.
- 노트 미리보기(peek) 팝업이 열린 상태에서 해당 파일을 삭제하면 팝업도 함께 닫힙니다.

## [0.7.7] - 2026-06-22

- 파인더에서 파일을 추가·삭제·이동한 뒤 앱으로 돌아오면 **사이드바가 자동으로 업데이트**됩니다.

## [0.7.6] - 2026-06-22

- **노트를 삭제해도 탭이 안 닫히거나 사이드바에 남아있던** 문제를 고쳤습니다.
- **이미 삭제된 노트**를 열려 할 때 탭이 그대로 남아있던 문제를 고쳤습니다.

## [0.7.5] - 2026-06-22

- 노트 **정보 패널의 목차에서 한글이 깨져 보이던** 문제를 고쳤습니다.
- **링크로 노트 만들기**를 할 때 동일한 노트가 **여러 개 생성되던** 문제를 고쳤습니다.

## [0.7.4] - 2026-06-17

- 노트 표의 **셀 안에서 줄바꿈(Enter)을 하면 표가 사라지던** 문제를 고쳤습니다. 이제 표가 그대로 유지됩니다.
- **체크리스트**가 다른 마크다운 앱에서도 깔끔하게 보이도록 정리했습니다.

## [0.7.3] - 2026-06-17

- 노드를 **여러 개 선택하고 우클릭**하면, 여러 개에 한 번에 쓸 수 있는 기능만 깔끔하게 보여줍니다.
- **색상**을 선택한 노드 **전체에 한 번에** 적용합니다. (완료 표시·삭제도 선택 전체에 적용)

## [0.7.2] - 2026-06-17

- **"업데이트 확인"을 누르면 바로** 진행 상황을 알려드립니다 — 확인 중 · 새 버전 받는 중 · 준비 완료까지 한눈에.
- 업데이트 내역을 더 읽기 쉽게 다듬었습니다.

## [0.7.1] - 2026-06-17

- 사이드바 아래 **＋** 버튼으로 새로 만들기가 다시 잘 동작합니다.

## [0.7.0] - 2026-06-16

- 🔍 **한 번에 전체 검색** — 모든 지도의 노드와 모든 노트를 제목·내용까지 찾아 곧바로 이동합니다. (⌘⇧F 또는 사이드바 검색)
- 🧹 **더 단순해진 화면** — 사이드바는 검색·라이브러리 중심으로 가벼워지고, 만들기는 ＋ 하나로, 설정은 ⚙(⌘,) 하나로 모았습니다.
- 사용법과 단축키는 설정의 **"사용 안내"**에 깔끔하게 정리했습니다.
- 가운데 주제(루트)도 끌어서 다른 항목 아래로 넣을 수 있습니다.

## [0.6.1] - 2026-06-16

- 탭을 끌어 순서를 바꾸기가 더 정확하고 매끄러워졌습니다.

## [0.6.0] - 2026-06-16

- 🔗 **노트끼리 잇기** — 글 속에서 `[[제목]]`으로 다른 노트를 연결합니다. 누르면 미리보기로 떠서 보던 흐름을 잃지 않고, 어떤 노트가 이 노트를 가리키는지(백링크)도 한눈에 봅니다.

## [0.5.3] - 2026-06-15

- 긴 노트 맨 위에 **목차**가 생겨 원하는 곳으로 바로 이동합니다.

## [0.5.2] - 2026-06-15

- 업데이트되면 **무엇이 좋아졌는지** 카드로 알려드립니다.
- 노트 표에서 행·열을 추가·삭제할 수 있습니다.

## [0.5.0] - 2026-06-12

- ⏱ **집중 세션** — 노드에서 타이머로 집중하고 목표·결과를 기록합니다.
- **"오늘"**과 **"돌아보기"**로 흩어진 일정을 모아 보고 한 주를 회고합니다.
- 노트에 **표**를 넣을 수 있습니다.

## [0.4.0] - 2026-06-12

- 노트에 **이미지**를 붙여넣기·드래그로 넣을 수 있습니다.

## [0.3.0] - 2026-06-11

- 새 버전이 나오면 **자동으로 받아** 알려드립니다.

## [0.2.0] - 2026-06-11

- 📝 **노트** — 마인드맵 곁에 자유로운 메모를 적고, 노드와 연결합니다.
