# MindMap

키보드 중심의 macOS 마인드맵 앱. Electron + React + TypeScript, d3-hierarchy 자동 정렬, Notion 디자인 시스템.

## 개발

```bash
npm install
npm run dev      # Vite 개발 서버 + Electron 창
```

## 빌드 / 패키징

```bash
npm run build    # 타입체크 + 렌더러/일렉트론 번들
npm run dist      # .dmg 생성 (release/)
```

## 키보드

| 키 | 동작 |
|---|---|
| `Enter` | 형제 노드 생성 |
| `Tab` | 자식 노드 생성 |
| `Space` / `F2` / 더블클릭 | 편집 시작 |
| `Enter` / `Esc` | 편집 종료 / 취소 |
| `Delete` / `Backspace` | 노드 삭제 |
| 방향키 | 노드 간 이동 |
| `⌘Z` / `⌘⇧Z` | 실행 취소 / 다시 실행 |
| `⌘N` / `⌘O` / `⌘S` / `⌘⇧S` | 새로 / 열기 / 저장 / 다른 이름으로 |
| `⌘+` / `⌘−` / `⌘0` | 확대 / 축소 / 화면 맞춤 |

- 노드를 **드래그**해 다른 노드에 가까이 놓으면 그 아래(자식)로 편입되고 자동 재정렬됩니다.
- 빈 캔버스 드래그로 화면 이동(pan), 트랙패드 두 손가락으로 이동, `⌘`+휠(핀치)로 확대/축소.
- 자식이 있는 노드 오른쪽의 `−`/`+` 버튼으로 접기/펼치기.
- 하단 툴바의 **색상**으로 선택 노드에 스티커 색을 칠합니다.

## 파일 포맷

- 기본 저장: `.mind` (JSON). 파일이 있으면 1.2초 디바운스 자동 저장.
- 가져오기/내보내기: Markdown(들여쓰기 불릿), OPML. (File ▸ Import / Export)
- 한 캔버스에 여러 루트(중심 토픽)를 둘 수 있습니다.
