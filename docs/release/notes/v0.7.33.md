# v0.7.33 — 2026-07-10

## 새 기능

- **노트 템플릿.** 회의록·회고처럼 자주 쓰는 노트 구조를 템플릿으로 저장해두면, 다른 노트를 쓸 때 편집기 툴바의 "템플릿+" 버튼(검색 가능)으로 커서 위치에 바로 불러올 수 있습니다. 템플릿은 사이드바 하단의 "Note Template" 버튼에서 만들고 관리하며, 삭제해도 자동으로 다시 만들어집니다. 설정에서 이 기능을 완전히 끌 수도 있습니다.
- 노트 편집기의 "메타+" 버튼에도 검색창이 추가되어, 항목이 많아져도 빠르게 찾을 수 있습니다.

## 보안
