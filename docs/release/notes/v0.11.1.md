# v0.11.1 — 2026-07-17

0.11.0의 캘린더·할 일 기능을 실제로 써보며 다듬은 릴리즈다. **집중 세션 안정화**(데이터 유실 수정)와
**캘린더 UX 일관화**가 핵심.

## 수정 — 집중 세션이 안전해졌어요

- **끝내기 직전에 적은 내용이 사라지지 않아요.** 집중을 끝내기 바로 전에 작업 로그를 적으면 저장이
  엇갈려 내용이 사라지거나, 세션이 "진행 중"으로 멈춰 기록에 안 남던 문제를 고쳤습니다.
- **멈춰 있던 세션 정리.** 예전에 이 문제로 "진행 중"에 멈춘 노트는 배너의 **"종료 처리"**로 마무리할 수 있어요.

## 새 기능 — 집중과 일정

- **일정 없이도 바로 집중.** 할 일로 바꾼 노드는 일정을 잡지 않아도 집중을 시작할 수 있습니다(일정 지정과
  집중은 별개예요).
- **집중한 시각이 캘린더에 자동으로.** 일정 없는 노드에서 집중을 시작하면 그 시각이 스케줄로 잡혀 캘린더에
  바로 뜨고, 끝나면 실제 집중한 만큼 블록 크기가 맞춰집니다.

## 캘린더

- **일정 추가가 어디서나 똑같아요.** 일·주·월 보기 모두 위쪽 **"일정 추가"** 버튼 하나로 통일. 종일/시간
  선택과 Esc 닫기도 일관됩니다.
- **일정을 누르면 내용을 먼저 보여줘요.** 지도를 바로 열지 않고 그 노드와 아래 내용을 잘림 없이 미리보고,
  다시 누르면 오른쪽에 지도가 열립니다.
- **월·주는 조망, 일간은 실행.** 월·주 보기에서 집중·완료 버튼을 정리해 "이번 달·이번 주에 뭐가 있나"를
  보는 데 집중했습니다.
- **겹치는 일정이 읽기 쉬워졌어요.** 시간이 겹치면 나란히 나뉘어 보이고, 마우스를 올리면 그 일정이 넓게
  펼쳐져 온전히 읽힙니다.
- **월간 보기가 다 보여요.** "+3" 같은 축약 없이 셀 안에서 스크롤로 전부 확인합니다. 주간 시간표가 시간
  눈금과 어긋나던 문제도 고쳤습니다.

## 정리 — 생각과 실행의 경계

- 완료·일정·집중 같은 실행 기능이 **할 일 노드에서만** 일관되게 보이도록 다듬었습니다(명령 팔레트와
  집중시간 표시 포함). 순수 생각 노드는 가볍게 유지됩니다.

## 주의

- 반복 일정은 아직 지원하지 않습니다(다음 단계).
- 닫혀 있는 지도의 노드에 캘린더에서 일정을 잡으면, 그 지도가 배경에서 함께 열립니다(안내가 뜹니다).
