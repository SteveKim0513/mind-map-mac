# v0.7.3 — 2026-06-17

## 버그 수정 · 개선

- **다중 선택 우클릭 메뉴 정리** — 노드를 여러 개 선택하고 우클릭하면 이제 선택 전체에 적용되는 기능만 보입니다("N개 선택됨" 헤더로 단일 메뉴와 구분): 두 노드 연결 · 섹션으로 묶기 · 색상 · 완료 · 삭제. 단일 노드에만 의미 있던 항목(편집·자식/형제 추가·메모·링크·노트·스케줄·집중·보기 등)은 다중 메뉴에서 숨김.
- **색상 일괄 적용** — 스와치를 누르면 선택한 노드 전체가 한 번에 바뀝니다. 완료 표시/해제·삭제도 선택 전체에 적용(삭제는 키보드 Delete와 동일하게 동작 — 기존엔 메뉴가 한 노드만 지우던 불일치를 고침).

## 검증

- E2E: 다중 선택 우클릭 → "2개 선택됨" + 적용 가능한 항목만 노출(단일 전용 숨김), 색상 스와치 → 선택 노드 모두 적용. 단위 테스트 69개 통과.
