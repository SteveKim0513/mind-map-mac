# v0.9.1 — 2026-07-15

## 새 기능

- **캘린더 탭.** 사이드바 "오늘"이 일/주/월 보기가 되는 진짜 캘린더 탭으로 승격됐다. 일간 뷰는 오늘 일정부터, 주간은 7일 그리드, 월간은 전통적인 달력 그리드(일정 개수 배지)로 보여주고, 어느 뷰에서든 칸을 클릭하면 그 날짜의 일간 뷰로 전환된다. 사이드바 상단은 "캘린더·즐겨찾기" 2개만 남기고, 집중 기록은 하단 관리 화면 묶음으로, 최근 수정은 설정(⌘,) 안으로 옮겼다.

## 개선

- **집중 세션은 일정이 지정된 노드에서만 시작할 수 있다.** 실행은 정리된 것 중 일정이 잡힌 걸 완수하는 흐름이라는 정의에 맞춰, 일정 없는 노드는 선택 툴바에 집중 버튼이 안 보이고 우클릭 메뉴는 비활성 + 안내 문구를 보여준다.
- **우클릭 메뉴가 구조/정리/실행/통찰 4그룹으로 재편됐다.**
- **일정 팝오버에 "지금 집중 시작" 버튼이 생겼다.** 날짜·시간을 정한 그 화면에서 바로 집중 세션으로 넘어갈 수 있다.

## 버그 수정
