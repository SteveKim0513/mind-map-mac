# v0.8.0 — 2026-07-13

## 새 기능

- **어디서든 ⌥Space로 빠르게 생각을 적으세요.** 앱을 열지 않아도 전역 단축키 한 번이면 작은 입력창이 뜨고, Enter를 누르면 "오늘의 생각"에 바로 쌓입니다. ⌘K "빠른 캡처 열기"로도 열 수 있어요.
- **"내일 3시", "#파랑" 같은 문구를 그냥 치면 알아서 정리됩니다.** 날짜·요일이 담긴 문구를 적으면 일정이 자동으로 잡히고, `#태그명`을 적으면 색이 자동으로 붙어요.
- **사이드바에 "최근 수정"·"즐겨찾기"가 생겼습니다.** 최근에 고친 파일을 바로 찾거나, 파일에 마우스를 올려 별표를 눌러 즐겨찾기에 모아둘 수 있어요.
- **⌘K가 자주 쓰는 명령을 기억합니다.** 같은 명령을 몇 번 실행하면 목록 맨 위로 올라오고 단축키(⌥1~9)가 자동으로 붙어요.

## 개선

- **설정 화면이 더 단순해졌습니다.** 테마·워크스페이스·노트 템플릿만 기본으로 보이고, 나머지(AI·정보 양식·사용 안내·업데이트)는 "고급 설정 보기"를 눌러야 나타나요.
- 집중 위젯이 평소엔 작은 타이머만 보이도록 접히고, 클릭하면 펼쳐집니다.
- 휴지통·Note Template 아이콘의 숫자 배지가 조용한 점 표시로 바뀌었습니다.
- 노트 편집기의 "메타+" 버튼과 설정의 "메타 템플릿"이 각각 "양식+"/"정보 양식"으로 이름이 바뀌었습니다.
- 우클릭 메뉴 등 플로팅 UI의 그림자·등장 애니메이션을 정리했습니다.
