# v0.4.1 — 2026-06-12

미리알림 동기화와 노트 편집의 자잘한 문제를 고친 패치.

## 버그 수정

- **미리알림 동기화** — 앱과 미리알림에서 서로 다른 항목(예: 앱에서 완료, 미리알림에서 제목)을 바꿨을 때 한쪽 변경이 사라질 수 있던 문제를 고쳤습니다. 이제 제목·날짜·완료 상태를 각각 따로 맞춰서 양쪽 변경이 모두 보존됩니다.
- **노트 슬래시 메뉴(`/`)** — 메뉴가 떠 있는 상태에서 화면을 스크롤하면 메뉴가 허공에 남던 문제를 고쳤습니다. 이제 스크롤하면 닫히고, 화면 아래쪽에서는 위로 펼쳐집니다.
