# v0.7.9 — 2026-06-23

## 버그 수정
- **자동 업데이트가 멈추던 문제 (회귀, v0.7.5–0.7.7)** — 일부 빌드가 자신을 "개발 빌드"로 잘못 인식해 "업데이트 확인"이 비활성화되고, 한 번 업데이트한 뒤로는 더 이상 새 버전을 받지 못하던 문제. 원인은 패키지 asar의 `productName` 누락으로 `app.getName()`이 `mind-map`으로 떨어진 것.
  - `build.extraMetadata.productName`을 빌드 설정에 영구 추가(모든 빌드가 productName을 asar에 주입).
  - `isUpdateEnabled()`를 `isPackaged && !getName().endsWith('Dev')`로 견고화(prod fail-open).

## 알려진 이슈
- **이미 0.7.5–0.7.7에 막혀 있는 설치본은 자동 업데이트가 불가**하므로, 0.7.9 dmg로 한 번 수동 재설치해야 한다. 이후 업데이트는 자동으로 동작.
- 빌드는 여전히 arm64 전용(Intel 미지원).
