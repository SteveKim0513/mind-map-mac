# v0.2.1 — 2026-06-11

외부 코드 리뷰에서 확인된 결함 7건을 수정한 패치 릴리즈. 특히 노트가 사라져 보일 수 있는 문제(아래 첫 항목)가 포함되어 있어 v0.2.0 사용자는 업데이트를 권장합니다.

**이 버전부터 Apple 공증된 서명 빌드입니다** — 다른 Mac에서도 "손상됨" 경고 없이 더블클릭으로 설치·실행됩니다.

## 버그 수정

- **동명 노트가 있는 폴더로 노트를 이동/승격하면 노트가 사라져 보이던 문제** — 이제 "이름 2.md"로 자동 개명되어 사이드바에 그대로 보입니다. (기존에 사라진 노트는 워크스페이스 폴더에 확장자 없는 파일로 남아 있으니 Finder에서 이름에 `.md`를 붙이면 복구됩니다)
- 노드 복사(⌘C/⌘V)가 링크 여러 개와 일정 날짜를 버리던 문제 — 이제 내용은 전부 복사되고, 미리알림 연동만 사본에서 꺼진 상태로 시작합니다
- 날짜만 있는 일정(빠른 칩 "오늘"·"내일" 등)이 마감 당일 내내 "지남(빨강)"으로 표시되던 문제 — 당일에는 '오늘'로, 날이 지나야 빨강
- "링크 추가"로 단 URL이 검색(⌘F)에 안 잡히던 문제
- 메모 타이핑 중 줄바꿈마다 카드가 출렁이던 문제
- 노트의 노드 칩 클릭이 (맵 파일이 이름 변경된 뒤) 조용히 무반응이던 문제 — 문서 id로 다시 찾아 열고, 못 찾으면 안내 토스트
- 노드를 드래그한 직후 방향키로 이동하면 선택 툴바가 계속 숨어 있던 문제
