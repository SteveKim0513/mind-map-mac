# v0.6.0 — 2026-06-16

## 새 기능

- **노트 ↔ 노트 연결** — 본문에서 `[[노트 제목]]`으로 노트를 잇습니다.
  - `[[` 입력 시 자동완성, 없는 노트는 그 자리에서 만들어 연결.
  - 링크 클릭 = **미리보기(peek)**(보던 노트 유지) → "열기" 또는 **⌘-클릭** = 반대쪽 화면에 나란히.
  - 없는/삭제된 대상으로의 링크는 흐리게 표시(클릭하면 생성).
  - 저장은 깨끗한 `[[ ]]` 마크다운(다른 에디터에서도 열림).
- **백링크(양방향)** — "정보"에 *이 노트를 참조하는 노트들*. 한쪽만 링크해도 양쪽에서 오감.
- **노트 "정보" 패널** — 목차 · 연결된 노드 · 백링크를 제목 옆 버튼 하나로(기본 닫힘).
- **노드 형제 순서 드래그** — 끌어서 순서 변경(틈=형제 삽입, 가운데=자식).

## 개선

- **탭 순서 드래그** — 탭을 끌어 재정렬·좌우 이동(포인터 기반으로 재구현, 타이틀바에서도 동작).

## 버그 수정

- 노드 → "새 노트 만들어 연결" 무음 실패(에러 노출 + 단계별 격리).
- 노트 이름 변경 시 다른 노트의 `[[ ]]` 일괄 갱신(부분일치 미오염).

## 검증

- 전 기능 격리 빌드 앱에서 E2E 검증(탭/노드 드래그, peek·옆에 열기 좌·우, 이름변경 치환, 백링크 라이브, 흐린 칩 동적 갱신, 목차 점프). 단위 테스트 69개 통과.

## 알려진 이슈 / 의도적 제외

- 위키링크 **호버 미리보기**는 넣지 않음(클릭=peek의 차분함 유지).
