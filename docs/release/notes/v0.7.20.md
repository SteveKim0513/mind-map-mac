# v0.7.20 — 2026-07-02

## 버그 수정

- **불릿/체크리스트에서 탭 키가 들여쓰기 대신 탭 문자를 삽입하던 문제** — 기존 내용이 있는 목록 항목에서 Tab을 누르면 탭 문자가 입력되던 현상을 수정했습니다. 이제 목록 안에서는 Tab이 항상 들여쓰기, Shift-Tab이 내어쓰기로 동작합니다.

## 신규

- **사이드바에서 파일 여러 개를 선택해 폴더로 한 번에 이동** — 파일을 여러 개 선택(클릭·Shift 클릭·Cmd 클릭)한 뒤 폴더 위로 드래그하면 선택된 파일이 모두 이동됩니다.

## 개선

- **툴바 블록 타입 전환이 모든 경우에 동작** — 불릿 포인트 영역에서 인용(blockquote)이나 다른 블록 타입으로 전환할 때 변환이 되지 않던 문제를 수정했습니다. 이제 어떤 블록 타입에서도 툴바로 타입을 바꿀 수 있습니다.
