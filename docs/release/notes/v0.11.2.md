# v0.11.2 — 2026-07-17

e2e에 커버되지 않던 사용자 journey를 실제 앱 구동으로 탐색해 발견한 47건을 수정하고,
회귀 그물망이 필요한 시나리오를 테스트에 반영한 안정화 릴리즈.

## 새 기능
- 없음 (안정화·버그 수정 중심 릴리즈)

## 개선
- **명시적 자정 유지** — `@오전 12시`처럼 자정을 지정하면 종일이 아니라 그 시각 일정으로 잡힌다 (`MindNode.allDay` 필드, ADR 0017).
- **이미지 이식성** — 노트 이미지 참조를 표준 마크다운 뷰어(Obsidian/GitHub/Typora)에서 열리도록 URL 인코딩. 앱 내 렌더링은 그대로.
- 붙여넣기가 할 일·시간블록(durationMin) 유지, 전역 검색이 노드 메모·링크도 검색, 학습된 ⌥숫자 퀵키가 순위 변동에도 고정.
- 내보내기 시 손실 경고, "화면 다시 맞춤" 라벨 명확화, "이번 주" 기준을 캘린더·집중 기록에서 통일.

## 버그 수정
- **버전 복원 시 현재 작업 유실** — 복원 직전 현재 내용을 무조건 스냅샷해 기록에 보존(5분 스로틀 우회).
- **표 파괴** — 표 셀의 `|`가 저장 후 재열기 시 표 전체를 무너뜨리던 문제 수정. 표 안에서 Tab으로 칸 이동도 가능.
- **열린 노트 외부 변경 덮어쓰기** — 다른 앱/기기에서 바뀐 노트를 조용히 덮어쓰지 않고 경고 후 재로드. 포커스 유지 중 외부 변경도 복구 가능하게 보관.
- **이름변경 시 버전 기록 고아** — `.history` 폴더가 새 이름을 따라 이동.
- **복제 시 미리알림 중복 예약** — 복제 노드가 리마인더 4개 필드를 모두 제거(복사와 동일, ADR 0016).
- **일정 해제가 캘린더에 반영 안 됨** — 해제 즉시 캘린더에서 사라짐.
- 워크스페이스 전환 시 설정 초기화, 이름변경 후 최근 파일 오류(ENOENT), `@내일 3시간`이 03:00으로 잡히던 파싱 오류, 서브트리 집중 중 색상 필터 스코프 무시, URL 가져오기 무한 로딩 등 수정.

## 주의
- 같은 파일 id의 노트가 다른 기기에서 동기화되면, 덮어쓰기는 막지만 열린 편집기 화면은 재오픈 전까지 갱신되지 않을 수 있음.
- 외부 변경 내용이 자동저장에 덮인 경우 기록에서 복구 가능하나, 별도 알림 토스트는 아직 없음.
