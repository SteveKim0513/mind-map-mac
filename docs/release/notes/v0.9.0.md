# v0.9.0 — 2026-07-15

## 새 기능

- **커맨드 팔레트(⌘K)에서 선택한 노드에 바로 동작을 실행할 수 있습니다.** 지금까지는 노드를 검색해 선택·포커스만 시켰는데, 이제 "선택 노드: 일정 설정" · "색상 — 보라" · "링크 추가" · "노드에 노트 연결" · "집중 세션 시작" 같은 실행 가능한 명령이 함께 뜹니다. 즐겨찾기·최근 수정 보기도 팔레트 명령으로 등록했습니다.
- **사이드바 파일/폴더 행에 우클릭 메뉴가 생겼습니다.** 즐겨찾기·이름 변경·삭제를 캔버스 노드·탭과 같은 방식(우클릭)으로도 실행할 수 있습니다. 호버 아이콘은 그대로 있습니다.

## 개선

- **선택 툴바가 6개 버튼(색상·완료·메모·집중·자식추가·삭제)으로 정리됐습니다.** 일정·링크·노드에 노트 연결은 값이 있을 때 나타나는 칩과 우클릭 메뉴로만 접근합니다.
- **색상 선택 UI를 하나로 통일했습니다.** 우클릭 메뉴와 선택 툴바가 각자 다른 구현을 갖고 있어 선택 툴바 쪽은 이미 지정한 색을 다시 열어도 표시가 안 됐는데, 이제 어디서 열어도 같은 모양·같은 표시입니다.
- 접기 화살표·링크 제거 버튼이 평소에도 옅게 보입니다. 노트 에디터 툴바도 그룹별로 시각적으로 구분됩니다.
- 탭 우클릭 메뉴, "템플릿 추가" 드롭다운이 이제 Escape로 닫힙니다.
- 정보 양식 삭제 시 확인을 먼저 받습니다 — 그 양식을 쓰는 모든 노트에서 함께 사라지는 되돌릴 수 없는 동작이라서요.
- 영어로 남아있던 라벨, 혼용되던 용어("템플릿"/"양식"), 원인·다음 행동이 없던 오류 메시지 여러 건을 정리했습니다.

## 버그 수정

- **탭을 편집 직후 바로 닫아도 방금 쓴 내용이 사라지지 않습니다.** 편집 중이던 탭은 닫기 전에 즉시 저장하도록 고쳤습니다.
