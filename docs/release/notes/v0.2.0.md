# v0.2.0 — 2026-06-11

첫 정식 릴리즈 사이클. 노트 시스템이 합류했고, 리마인더 동기화의 핵심 결함을 잡았다.

## 새 기능

- **노트**: 독립 Markdown 파일 기반 노트 (TipTap 에디터, `/` 블록 메뉴, 체크리스트). 노드와 양방향 연결 — 노드 칩 → 프리뷰 팝업 → 수정/연동 해제, 노트 → 노드 점프. URL로 노트 만들기(본문 자동 추출).
- **"제목 없음" 자동 이름**: 이름 없이 만든 맵은 첫 중심 토픽 텍스트가 파일 이름이 된다.
- **시맨틱 색상 시스템**: 색을 의미 키로 저장 — 팔레트·다크 모드 개선이 기존 파일에 자동 적용.
- 보기 메뉴에 화면 분할 토글 추가.
- **앱 아이콘** — 앱의 노드 언어(잉크 루트 → 태그 색 가지)를 그대로 담은 첫 정식 아이콘.

## 개선

- **리마인더 동기화 (중요)**: 인바운드(미리알림 → 앱) 동기화가 폴링 쿼리 타임아웃으로 사실상 동작하지 않던 문제 수정 — 쿼리를 일괄 조회로 재작성 (23초 → 0.85초). 장애 시 재시도 백오프가 실제 복구를 기준으로 동작하도록 개선.
- 우클릭 메뉴를 5개 그룹(구조/속성/첨부/일정/보기)으로 구분 (decisions/0004).
- 노드 제목 저장 시 공백 정리.

## 버그 수정

- 노트 연결 픽커가 Esc로 닫히지 않고 전역 단축키(⌘K/⌘P)를 막던 문제
- 우클릭 메뉴가 하단 줌 툴바에 가려지던 문제
- 공백만 입력한 노드가 빈 모양으로 남던 문제

## 알려진 이슈

- 반복 일정 미지원 (일회성만)
- 시스템 시간대 변경 시 기존 일정 시각이 어긋날 수 있음
- 노트 파일을 직접 삭제하면 노드에 연결 흔적이 남음
