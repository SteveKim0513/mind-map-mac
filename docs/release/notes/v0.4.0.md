# v0.4.0 — 2026-06-12

노트 기능 강화. 제목이 파일명과 따로 놀던 문제를 고치고, 이미지를 넣을 수 있게 했습니다.

## 새 기능

- **노트 이미지** — 툴바 이미지 버튼, 붙여넣기, 드래그&드롭으로 이미지를 넣을 수 있습니다. 큰 이미지는 자동으로 적당한 크기로 줄여서 넣고, 노트 파일 하나에 함께 담기므로 다른 곳으로 옮겨도 이미지가 따라갑니다.

## 버그 수정

- **노트 제목을 바꿔도 사이드바·탭 이름이 그대로이던 문제** — 이제 제목을 편집하면 파일명과 목록이 함께 바뀝니다.

## 비고

- 자동 업데이트가 적용된 첫 후속 릴리즈입니다. v0.3.0 사용자는 앱이 자동으로 받아 "지금 재시동 / 나중에"를 안내합니다.

## 알려진 이슈

v0.3.x와 동일 (반복 일정 미지원 · 시간대 변경 시 일정 어긋남 가능 · 노트 파일 직접 삭제 시 연결 흔적 잔존)
