# v0.7.15 — 2026-06-24

## 버그 수정

이미지 삽입·관리 전반의 버그를 수정했습니다.

- **툴바로 삽입한 이미지가 외부 파일로 저장되지 않던 문제** — 붙여넣기·드래그 경로와 동일하게 `.assets` 폴더에 저장됩니다.
- **노트 미리보기 팝업에서 이미지가 보이지 않던 문제** — 팝업도 이미지를 정상적으로 표시합니다.
- **노트 이름 변경 후 이미지가 깨지던 문제** — `.assets` 폴더 이름도 함께 변경됩니다.
- **노트를 폴더 이동 후 이미지가 사라지던 문제** — `.assets` 폴더도 함께 이동합니다.
- **휴지통에 넣었다 복원하면 이미지가 없어지던 문제** — `.assets` 폴더도 함께 이동/복원/삭제됩니다.
- **같은 초에 이미지를 여러 장 붙여넣으면 파일이 겹치던 문제** — 밀리초 단위 파일명으로 해결됩니다.
- **이미지 보관 폴더(`.assets`)가 사이드바 파일 트리에 나타나던 문제** — 더 이상 표시되지 않습니다.
