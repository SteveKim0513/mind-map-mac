# v0.5.1 — 2026-06-14

## 버그 수정

- **노드에서 만든 노트가 사이드바에 보입니다.** 노드의 "노트 추가·연결 → 새 노트"로 만든 노트가 숨김 폴더로 들어가 사이드바에 안 나타나던 문제를 고쳤습니다. 이제 노드 노트는 **그 노드의 맵과 같은 폴더**에 생성됩니다.
- **기존에 숨어 있던 노드 노트도 자동으로 살아납니다** — 이전 버전에서 만든 노드 노트는 앱을 열 때 워크스페이스 루트로 옮겨져 사이드바에 나타납니다.

## 개선

- 사이드바에서 숨기는 건 **집중 세션(work-log) 노트뿐** — 사용자가 직접 만든 "work-log" 이름의 폴더는 더 이상 숨지 않습니다.

## 위치 규칙

- 노드에서 만든 노트 → 그 노드의 맵과 같은 폴더
- 그냥 "새 노트" → 현재 위치
