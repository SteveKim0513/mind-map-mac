# v0.10.1 — 2026-07-16

두 갈래를 함께 담았다: **눈에 보이지 않는 곳의 데이터 신뢰·안전 강화**(IF-01~14)와,
**오늘(일간) 보기 재설계 + 온보딩 정리**.

## 새 기능 — 안 보이는 곳의 안전 (IF-01~14)

- **저장이 더 안전해졌어요** — 저장 중 문제가 생겨도 파일이 깨지지 않도록 원자적으로 기록(IF-01), 앱이
  실수로 두 번 열리지 않게(IF-03), 외부에서 가져오는 주소는 사설망 접근을 막았습니다(IF-12).
- **파일 버전 기록** — 저장할 때마다 이전 버전이 자동으로 남아, ⌘K "버전 기록"에서 언제든 되돌릴 수
  있습니다(IF-02).
- **외부 변경 감지** — 열어둔 맵이 다른 곳(동기화 등)에서 바뀌면 다시 불러올지 안내합니다(IF-04).
- **죽은 링크 정리** — 노드를 지우거나 이름을 바꾸면, 그 노드를 가리키던 노트의 링크가 자동으로 정리·갱신
  됩니다(IF-05).
- **겹침 정돈** — ⌘K "겹침 정돈"으로 겹친 가지들을 아래로 정리합니다(IF-08, 되돌리기 가능).
- **시스템 테마 따라가기**(IF-09), **집중 중 자리비움 구분 기록**(IF-07), 검색·캘린더 **캐시로 더 빠르게**(IF-10).
- **휴지통 자동 정리(선택)** — 원하면 3개월 지난 휴지통 항목을 시스템 휴지통으로 자동 정리합니다.
  **기본은 꺼져 있어요** — 직접 켜야 동작하며, 정리해도 시스템 휴지통에서 복구할 수 있습니다(IF-06).

## 새 기능 — 오늘(일간) 보기

- **오늘 보기가 더 명확해졌어요** — 오늘 일정만 카드로 보여주고, 긴 제목도 잘리지 않아요. 다음에 할 일이
  강조되고 "집중"을 바로 시작할 수 있습니다. 지난 일정은 위쪽 "지남"에 접혀 있어요.

## 변경

- **첫 실행 안내(온보딩)를 잠시 내렸습니다** — 기능이 더 자리 잡은 뒤에 제대로 만들기 위해서예요([결정
  0013](../../decisions/0013-onboarding-after-features-stable.md)). 빈 화면에서는 그대로 `Enter`로 시작할 수 있습니다.

## 주의

- 반복 일정은 아직 지원하지 않습니다(다음 단계).
- IF-06 휴지통 자동 정리는 **기본 꺼짐**입니다 — 사용자 동의 없이 데이터를 지우지 않습니다.
