# v0.3.0 — 2026-06-11

자동 업데이트가 추가된 첫 버전. 이 버전부터는 새 버전이 나오면 앱이 알아서 받아서 알려줍니다.

## 새 기능

- **자동 업데이트** — 앱이 백그라운드에서 새 버전을 확인하고, 다운로드가 끝나면 "지금 재시동 / 나중에"를 안내합니다. 앱 메뉴 → "업데이트 확인…"으로 직접 확인도 가능합니다. (이제부터 dmg를 수동으로 다시 받을 필요가 없습니다)

## 비고

- v0.2.1의 모든 수정 포함 (코드 리뷰 7건 + 서명·공증).
- 자동 업데이트는 정식 설치본에서만 동작합니다.
