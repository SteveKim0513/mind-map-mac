# v0.10.0 — 2026-07-16

두 갈래의 큰 개선을 함께 담았다: **캘린더를 "실행하고 되돌아보는" 도구로 강화**(Phase 1-3)하고,
**핵심 기능의 접근점을 "물 흐르듯" 자연스럽게 정리**(카피 감사 Part 2)했다.

## 새 기능

### 캘린더 강화 (Phase 1-3)

- **드래그로 일정 재조정** — 주간·월간 보기에서 일정을 다른 날짜로 끌어다 놓으면 옮겨진다(주간 그리드는
  다른 시각까지). 전부 `mapStore.setScheduleAt` 경유라 `reminderOn` 노드는 미리알림 due도 함께 이동한다.
- **주간 시간 그리드** — 6–24시 시간대 세로축에 일정을 블록으로 배치. 종일·시간대 밖 항목은 상단 "종일" 띠에.
- **빈 슬롯 캡처** — 주간 그리드 빈 곳을 클릭하면 그 시각의 일정 노드를 열린 맵에 바로 만든다.
- **계획 ↔ 실행 대조** — 월간 셀·주간 헤더·일간 요약("계획 N · 집중 Xh · 완료 M")에 그날 실제 집중한 시간을
  계획과 나란히 보여준다. 같은 노드가 계획과 실행 기록을 동시에 갖는 이 앱만의 대조.
- **타임블로킹** — 일정에 소요 시간을 정하면(SchedulePopover의 "소요 시간" 또는 블록 하단 드래그) 시간표에
  그만큼 블록 길이로 표시된다. `durationMin`은 선택 필드이며 스키마 버전은 그대로다(하위호환, 결정 0012).

### 접근점 이동 (카피 감사 Part 2)

- **미리알림 동기화 진입점** — 설정 첫 화면의 상시 상태 행 + ⌘K "미리알림 동기화 설정"으로 바로 확인·연결.
- **⌘L** — 선택한 노드에 노트 연결. **⌘⇧N** — 새 노트(⌘N=새 마인드맵과 대칭).
- **첫 실행 안내(FirstRunCoach)** — 처음 빈 맵에서만 뜨는, 건너뛸 수 있는 3단계 안내(적기·잇기·일정). 한 번만.

## 개선

- **고객 언어로 문구 정리** — "집중 세션"→"집중", 우클릭 메뉴의 축 라벨(정리/실행/통찰) 노출 제거(구조는 유지),
  "리마인더"→"미리알림"으로 OS 용어와 통일, "포커스 해제"→"전체 보기".

## 주의

- 반복 일정은 아직 지원하지 않는다(스키마 변경이 필요해 다음 단계로 분리).
- 캘린더에서 직접 편집(드래그·캡처)은 현재 열려 있는 맵의 노드만 대상이다.
