# v0.7.35 — 2026-07-10

## 새 기능

- **⌘K로 노드까지 찾을 수 있습니다.** 명령 팔레트(⌘K)에 검색어를 입력하면 열려 있는 맵의 노드도 함께 나옵니다. 더 찾고 싶으면 하단의 "전체에서 찾기"를 눌러 노트까지 포함한 전체 검색으로 바로 이어갈 수 있어요. 파일 빠른 열기(⌘P)도 파일명으로 못 찾으면 같은 안내를 보여줍니다.
- **필요할 때 살짝 알려드립니다.** 탭을 많이 열었거나, 비슷한 노트를 여러 번 쓰거나, 한 맵에 일정이 늘어나면 화면 분할·노트 템플릿·리마인더 동기화를 한 번씩 제안해 드려요. 원치 않으면 닫으면 다시 뜨지 않습니다.

## 개선

- 사이드바 하단의 휴지통·Note Template 아이콘이 하나로 묶여 보이고, 설정 아이콘과는 분리됩니다.
- 노트 편집기 툴바의 "메타+"/"템플릿+" 버튼에 아이콘이 붙어 한눈에 구분됩니다.
- 노드 2개 이상 선택했을 때 뜨는 메뉴에도 "구조"/"속성" 그룹 표시가 추가됐습니다.
