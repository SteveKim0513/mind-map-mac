# v0.7.2 — 2026-06-17

## 개선

- **업데이트 확인 — 즉시 인앱 팝업** — "업데이트 확인"(설정 버튼/메뉴)을 누르면 곧바로 팝업이 뜨고 상태를 보여줍니다: 확인 중(스피너) → 최신 버전 ✓ / 새 버전 받는 중 N%(진행 바) / 준비 완료([지금 재시동][나중에]) / 실패([다시 시도]). 네트워크가 끝나야 뜨던 느린 네이티브 다이얼로그(=오류처럼 보이던 지연)를 대체.
- **변경 내역 분리** — 사용자용(`CHANGELOG.user.md`, 고객 언어·간결, 앱에 노출)과 개발용(`CHANGELOG.md`, 기술 상세)으로 나눔.

## 검증

- 업데이트 확인 → 150ms 안에 "확인 중…" 팝업, 결과로 전환(E2E). 앱 "업데이트 내역"이 사용자 문구만 노출(개발 용어 누출 0). 단위 테스트 69개 통과.
