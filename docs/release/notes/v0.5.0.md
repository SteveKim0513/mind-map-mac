# v0.5.0 — 2026-06-12

생각·기록·일정에 **실행(집중)** 축이 더해졌습니다. 노드에서 집중 세션을 시작해 목표·과정·결과를 남기고, "오늘"로 계획하고 "돌아보기"로 회고하는 한 루프가 완성됩니다.

## 새 기능

- **집중 세션** — 노드 우클릭/선택 툴바 → "집중 세션 시작". 시작 시 *목표 한 줄*(팝업)을 받고, 자동 생성된 작업 로그 노트에 *과정*을 기록하며, 종료 시 완료 카드에서 *결과*를 남깁니다. 목표→과정→결과가 구조화 데이터로 측정됩니다. 진행 중 타이머 pill, 노드에 누적 시간 칩.
- **"오늘" 뷰** (사이드바) — 여러 맵에 흩어진 일정을 지남/오늘/다가오는으로 한 화면에 모으고, **클릭 한 번에 집중 세션 시작**.
- **돌아보기** (옛 작업 기록) — 집중 기록을 **오늘 ↔ 이번 주** 두 스코프로 돌아보기. 큰 숫자 요약 → 주간 띠 → 무엇을 했나(성과) → 더보기(시간대·추세). 목표 대비 성과까지.
- **노트 표(테이블)** — GFM 마크다운 라운드트립.

## 개선

- **접근성(WCAG AA)** — 노트 편집기에 접근성 이름, 텍스트 대비 정비(사이드바·"저장됨"·옅은 텍스트). 자동 a11y 점검 도입.
- **디자인 일관성** — 상태색을 텍스트로 쓰던 곳 정리(아이콘/틴트로), 모달 모션·타이포 통일, 설정 패널 텍스트 크기 정렬.
- **사이드바 IA** — "오늘 · 돌아보기"를 상단 기본 내비로 분리(설정과 구분).
- **로깅** — 메인 프로세스 미처리 예외/거부도 로그에 기록(크래시 진단), 렌더러 에러 레벨 보존.

## 버그 수정

- 집중 세션 목표가 노트 자리표시자를 잘못 가져오던 문제(직렬화 마커 불일치) 수정.
- 완료 카드 회고가 Esc 시 사라지던 문제 — 이제 저장.
- 대시보드에서 아래 컨트롤이 위 정보를 바꿔 시야 밖이 되던 인터랙션 결함 수정(컨트롤·결과 인접 배치).

## 알려진 이슈

- 분할 모드에서 비포커스 쪽 탭은 의도적으로 흐려져 대비가 낮습니다(포커스 표시용, 의도된 예외).
- "오늘" 뷰의 인라인 완료·집중은 열린 맵에서 동작하며, 닫힌 맵은 먼저 열립니다(v1 한계).
