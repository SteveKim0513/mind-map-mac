# v0.7.25 — 2026-07-08

## 개선

- **사이드바가 더 직관적으로 바뀌었습니다.** 오늘·돌아보기가 파일 목록 위에 탐색 항목으로 올라오고, 마인드맵·노트·URL 가져오기·새 폴더 버튼이 라이브러리 헤더에 모였습니다. 자주 쓰는 기능을 한눈에 찾을 수 있습니다.
- **UI 문구를 더 자연스럽게 다듬었습니다.** "링크로 노트" → "URL에서 가져오기", "미설정" → "연결 안 됨", "연동 해제" → "연결 끊기", "완전 삭제" → "영구 삭제" 등 전반적인 용어를 정리했습니다.
- **API 키 입력 중 취소할 수 있습니다.** 키가 없는 상태에서 "연결"을 눌렀다가 마음이 바뀌어도 취소 버튼으로 되돌아올 수 있습니다.
