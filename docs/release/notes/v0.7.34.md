# v0.7.34 — 2026-07-10

## 버그 수정

- **노트 템플릿 탭·편집 화면이 일반 노트와 구분되지 않던 문제를 고쳤습니다.** 템플릿 탭은 전용 아이콘으로 표시되고, 편집 화면 제목 옆에 "템플릿" 배지가 항상 붙어 있습니다. 여러 탭을 열어둬도 어떤 게 템플릿인지 바로 알 수 있습니다.
- **템플릿 생성 진입점 2곳의 동작을 통일했습니다.** 툴바의 "템플릿+ → 새 템플릿 만들기"가 이전엔 조용히 파일만 만들고 사이드바에서 찾으라는 안내만 띄웠는데, 이제 사이드바 패널에서 만들 때처럼 즉시 편집 화면이 열립니다. 새 템플릿의 기본 제목도 일반 노트와 겹치지 않도록 "새 템플릿"으로 바꿨습니다.
