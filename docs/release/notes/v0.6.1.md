# v0.6.1 — 2026-06-16

> v0.6.0의 탭 드래그 UX 다듬기 (사용자 피드백: 분할 가이드가 너무 크고, 순서 조정이 어렵다).

## 개선

- **탭 순서 조정** — 탭 사이 틈에 또렷한 세로 삽입 막대를 표시하고, 커서가 탭의 왼/오른쪽 절반 어디에 있는지로 앞·뒤 삽입을 결정 → 원하는 위치에 정확히 놓입니다(기존 희미한 inset 표시 + 항상 앞-삽입 → 개선).
- **화면 분할 가이드** — 탭을 **패널 영역으로 끌어내렸을 때만** 표시(스트립 내 순서 변경 시엔 미표시). 큰 점선 박스 → 작은 알약 라벨 + 연한 틸트로 경량화, 가리키는 쪽 강조 복원.

## 검증

- E2E: 스트립 내 드래그 시 가이드 미표시 / 패널 위로 가면 표시, 중점 기준 앞·뒤 삽입 정밀도, 삽입 막대 표시. 단위 테스트 통과.
