# v0.7.4 — 2026-06-17

## 버그 수정

- **노트 표 소실** — 표 셀 안에서 줄바꿈(Enter)을 하면(셀에 블록이 2개 이상) 저장·탭전환 시 표가 통째로 `[table]` 한 단어로 바뀌어 사라지던 문제를 고쳤습니다. 이제 어떤 셀 내용이든 항상 유효한 GFM 표로 저장되어 보존됩니다(여러 줄 셀은 한 줄로 합쳐짐 — GFM 표 셀의 본래 제약).
- **체크리스트 정리** — task 항목 사이에 빈 줄이 들어가 "느슨한" 목록으로 저장되던 것을 표준 GFM 체크리스트(`- [ ] a` 연속)로 다듬어, 다른 마크다운 앱에서도 깔끔하게 보입니다. 앱 화면·라운드트립은 동일.

## 검증

- 모든 블록 종류(제목·마크·목록·체크리스트·인용·코드블록·구분선·표·혼합)를 라운드트립 검증 → 표 외 다른 `[placeholder]`-류 손실 없음. 표 셀 Enter / 체크리스트 정리 E2E 통과. 단위 테스트 69개 통과.
