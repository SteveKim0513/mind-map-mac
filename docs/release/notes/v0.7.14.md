# v0.7.14 — 2026-06-24

## 개선

- **노트 이미지 화질 개선** — 최대 해상도 1600px → 2400px, JPEG 압축 품질 0.82 → 0.92. 흐릿하게 보이던 이미지가 훨씬 선명해집니다.
- **이미지 클릭 시 크게 보기** — 노트 안의 이미지를 클릭하면 전체 화면으로 확대됩니다. ESC 또는 바깥 클릭으로 닫습니다.
- **이미지를 별도 파일로 저장** — 삽입한 이미지가 `노트이름.assets/` 폴더에 `image-날짜-시각.jpg` 형식으로 저장됩니다. 마크다운 파일이 작아지고 다른 에디터(Obsidian 등)에서 열 때도 이미지가 정상 표시됩니다.
