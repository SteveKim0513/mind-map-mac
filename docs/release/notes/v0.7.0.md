# v0.7.0 — 2026-06-16

## 새 기능

- **전체 검색** — 워크스페이스 전체를 한 번에: 모든 지도의 노드 + 모든 노트의 제목·내용.
  - 결과 클릭 → 노드는 지도를 열어 가운데로, 노트는 바로 엽니다. 노트는 본문에서 걸리면 스니펫 표시.
  - 열기: **⌘⇧F** · 사이드바 검색 칸 · 노트·홈에서 **⌘F** · ⌘K "전체 검색".

## 개선

- **사이드바 재구성(애플식, 콘텐츠 중심)** — 검색 → 오늘/돌아보기(스마트 항목) → 라이브러리 트리 → 풋터(＋ · ⚙). 항상 떠 있던 만들기 4버튼과 인라인 설정 패널 제거.
  - 만들기 = 풋터 **＋ 하나** → 마인드맵/노트/링크로 노트/새 폴더 팝오버.
- **설정 = 오버레이(⚙ / ⌘,), 진짜 옵션만** — 테마 · 글자 크기 · 워크스페이스 폴더 · 버전·업데이트.
- **매뉴얼** — 설정의 "사용 안내" → 섹션 레일이 있는 팝업(시작하기 · 단축키 · 노트·연결 · 집중·회고). 단축키 리스트는 여기로 이동.

## 버그 수정

- 루트 노드를 끌어 다른 노드의 **자식으로** 만들 수 있습니다(빈 곳에 놓으면 기존처럼 자유 배치).

## 검증

- 전역 검색(노드·노트·본문 매칭, 노드/노트 이동), 루트 reparent, 새 사이드바·설정·매뉴얼 4화면을 격리 빌드 앱에서 E2E·스크린샷 확인. 단위 테스트 69개 통과.
