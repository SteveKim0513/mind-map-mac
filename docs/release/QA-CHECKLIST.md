# 배포 전 QA 체크리스트

> 원천: [FEATURE-INVENTORY.md](../product/FEATURE-INVENTORY.md). 기능이 추가되면 여기에도 시나리오를 추가한다.
> 환경: 패키징된 빌드(.dmg 설치본) 기준. 개발 모드에서만 통과하는 것은 통과가 아니다.

## 여정 1 — 맵 작성 (핵심 루프)

- [ ] 새 마인드맵 생성 → 키보드만으로(Tab/Enter/방향키) 3단계 깊이 맵 작성
- [ ] 노드 드래그로 부모 변경, Alt+↑↓로 형제 순서 변경
- [ ] 색·아이콘·완료 지정, 접기/펼치기, `Z`로 서브트리 확대
- [ ] `⌘Z` 연타로 전부 되돌리고 `⌘⇧Z`로 복원 — 구조·속성 모두 복원되는가
- [ ] 앱 강제 종료 후 재실행 → 마지막 1초 디바운스 내 변경 외 데이터 보존, 탭·분할 상태 복원

## 여정 2 — 일정·리마인더 (차별 기능, 가장 꼼꼼히)

- [ ] 노드에 내일 09:00 일정 설정 → 리마인더 앱 "MindMap" 리스트에 생성 확인
- [ ] 앱에서 노드 텍스트 수정 → 리마인더 제목 반영 (디바운스 후)
- [ ] 리마인더 앱에서 완료 체크 → 앱 노드에 완료 표시 (최대 45초)
- [ ] 일정 걸린 노드 삭제 → 리마인더도 삭제 / `⌘Z` 복구 → 다음 동기화에서 리마인더 재생성
- [ ] 자동화 권한 거부 상태에서 실행 → "권한 필요" 안내가 뜨고 앱이 멈추지 않는가

## 여정 3 — 노트 연동

- [ ] 노드 우클릭 → 새 노트 만들어 연결 → 오른쪽 분할에서 열림
- [ ] 슬래시 메뉴로 제목·체크리스트 작성 → 자동 저장 → 외부 에디터로 .md 열어 내용 확인
- [ ] 노드의 노트 칩 클릭 → 프리뷰 팝업 → "수정"으로 편집 진입
- [ ] 노트 칩에서 노드로 점프(연동 칩 클릭) → 해당 맵·노드 포커스
- [ ] URL로 노트 만들기 → 본문 추출 확인 (실패 시에도 빈 노트로 생성되는가)

## 여정 4 — 파일·셸

- [ ] Markdown/OPML 내보내기 → 다른 도구에서 열기 / 다시 가져오기 → 구조 보존
- [ ] 사이드바: 이름 변경(열린 탭 제목 갱신), 폴더 이동, 삭제(휴지통 + 탭 닫힘)
- [ ] `⌘F` 검색 순환, `⌘K` 명령, `⌘P` 파일 열기
- [ ] 다크 모드 전환 → 캔버스·노트·팝업 전부 적용되는가
- [ ] 워크스페이스 폴더 변경 → 트리 갱신, 기존 탭 정상

## 회귀 주의 지점 (과거 버그)

- [ ] 리마인더 동기화 중 앱이 행 걸리지 않는가 (osascript 직렬화 회귀)
- [ ] 같은 노드의 리마인더가 중복 생성되지 않는가 (reaping 회귀)
