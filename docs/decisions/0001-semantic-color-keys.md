# 0001. 노드 색은 hex가 아닌 시맨틱 키로 저장

날짜: 2026-06-10 (소급 기록) · 상태: 채택

## 맥락

노드 태그 색을 파일에 저장할 때 hex 값을 그대로 쓰면, 팔레트를 조정하거나 다크 모드 대응 색을 바꿀 때마다 기존 파일이 구식 색으로 남는다.

## 결정

문서에는 시맨틱 키('red'~'brown' 8종)만 저장하고, 렌더링 시 `src/theme/palette.ts`가 CSS 변수(`--tag-*`)로 해석한다. 레거시 hex는 로드 시 `normalizeColor()`로 자동 마이그레이션.

## 결과

팔레트 재조정·테마 추가 시 파일 수정 불필요. 대신 사용자 커스텀 색은 지원할 수 없는 구조 — 커스텀 색 요구가 생기면 이 결정을 다시 본다. 상세 규칙은 [design/COLOR-SYSTEM.md](../design/COLOR-SYSTEM.md).
